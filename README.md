### Como utilizar
Instalar dependencias
``` bash
$ npm install
```
Para construir (por ahora solo sass).
``` bash
$ npm run build
```
Para construir continuamente.
``` bash
$ npm run dev
```
Para construir y minimizar.
``` bash
$ npm run deploy
```
